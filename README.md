# 📝 Blogify - A Simple Blog Application

Live Preview 👉 [https://blog-1-o3cw.onrender.com]

---

## 📌 About the Project

**Blogify** is a full-stack blog application where users can:
- ✍️ Create, edit, and delete personal blogs
- 👥 Register and log in securely
- 🔐 View blogs by other users
- 🗨️ Comment on blogs (login required)
- ❤️ Like blogs (login required)

Built with simplicity and clean UI using **Node.js**, **Express.js**, **MongoDB**, and **EJS** templating engine.

---

## ⚙️ Tech Stack

- **Frontend:** EJS, HTML, CSS, Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Authentication:** JWT + Custom Middleware
- **Hosting:** Render

---

## 🚀 Features

- ✅ User registration and login
- ✅ Session-based user authentication
- ✅ Create, edit, and delete blog posts
- ✅ Add comments to blogs
- ✅ Like/unlike blogs
- ✅ Responsive UI
- ✅ Protected routes
